 <center><h2>SulinOS installation media</h2></center>
<br><center><a class="foolink" style="padding: 10px;  background: #de6581;  color:white; border-radius: 5px;" href="https://sourceforge.net/projects/sulinos/files/iso_images/"><span class="download">Look at stable <b>SulinOS</b> ISO Files.</span></a></center>
<br><center><br>You can write to your usb memory with <a href="https://etcher.io">Etcher</a> </br></center>
<br>SulinOS iso images are iso9096 format. You can use legacy and uefi bootloader.
SulinOS iso images may have bug. If you find any bug, you must open <a href="https://gitlab.com/groups/sulinos/-/issues">issue on gitlab</a>.
SulinOS is rolling operating system so do not have fixed version number. Please prefer latest iso images.
<br>
<br>SulinOS iso images creating with grub-mkrescue command and uses gpt partition table. 
If your devine not support booting on gpt partition table (for example old legacy bios), 
You should trying to write iso image with <a href="https://unetbootin.github.io/">unetbootin</a>. 
<br>Note: Live username is "<b>user</b>" and password is "<b>live</b>"
